Tools for New Programmers

Useful and fun tools to help you learn and code faster?
GitCommand LineVS CodeEmmetAlfredPock
In /code/tools
On June 20, 2020

InIn your path from beginner to hobbyist to professional programmer, you'll find that the right tool can make a huge difference in your productivity and growth. This is a list of things I found to be really helpful as I started my professional journey.

Code Editor

The most important resource as you are learning is a good editor. You can write code in any text-editor but one designed specifically for programming comes with many features that will make your life better, way better.

A code editor does many things you will immediately appreciate. Things like syntax highlighting,1 autocompletion,2 multiple-cursors, and much more. These can save a lot of time and help you learn.

Choosing the right editor is very much about what feels right to you. And, there are a lot of options. Popular choices include:

  • Visual Studio Code
  • Vim
  • Atom
  • Sublime

But, if you are just getting into coding, I recommend Visual Studio Code (usually just referred to as "VS Code"). It's an editor that's incredibly powerful and popular, but also friendly for beginners.

Programmers like to do everything on their keyboard with shortcuts because moving their hand to and from their mouse is slow.3 So most editors have bare-bones, stripped-down interfaces. VS Code's is more robust. There are good 'ol buttons and menus you can click on as you learn. But, it also has all of keyboard-based shortcuts and tools other editors offer. VS Code also has an extensive extension library with easy, one-click installs. This is awesome when you are beginning.

Finally, VS Code has excellent Git integration. If you've never used Git, then you'll appreciate how easy VS Code makes it to use. It provides a graphical interface for Git which is more intuitive than using the command line when you are learning. With that in mind...

Git

Git is a version control system, a sort of Google Docs for your code. It tracks the changes to your code ("diffs") each time you "commit" your files. It also allows you and others to branch your work off and then merge in those changes once they are done. This way you always have a working "main" branch and can develop new ideas in isolated "feature" branches.

Git it makes it easy to collaborate when paired with GitHub -- a separate service that is an online repository for codebases. Developers work on a project on their machine. Then "push" their changes to GitHub where other developers can review or "pull" their work. Git and GitHub's learning curve can be a little steep at first, but using a visual interface after a few tutorials should get you confortable enough to never need to save archived or versioned copies of files again.

Monospace Font with Ligatures

I didn't realize how much this mattered to me until I found a font I really liked. And, boy do I love JetBrains Mono!

Coding fonts are monospaced, which means that each character is the same width. This is crucial for coding where spacing is visually helpful and sometimes a requirement of the language. But, many monospaced fonts can be unpleasant to look at for eight hours a day.

Fonts designed for coding pay particular attention to symbols like &#96;, |, and => that aren't as common outside of code. And, they are specifically designed to make it easier to differentiate between easily confused letters and letter combinations. Many fonts accomplish this with ligatures4 which can help you read your code more easily.

Emmet

Emmet is a plugin for code editors (it comes preinstalled with VS Code) that helps you generate HTML content quickly. Once you get the hang of it, you'll save tons of time typing out annoying HTML syntax with classes, IDs, and more.

The syntax isn't too tricky to pick up, but it is extremely powerful. For example, here's a more complex command from the docs that generates HTML that would be a pain to type: #page>div.logo+ul#navigation>li*5>a{Item $}

Window Manager (macOS)

When coding, you'll often be moving back and forth between a few applications at a time. You'll want a terminal, an editor, and browser open all at once. You'll want these arranged nicely on your screen to avoid having to switch back and forth over-and-over.

MacOS has some built in features to help with this, but it's worth picking up a 3rd party option with a little more power especially if you have a larger external monitor. I use Magnet but there is also a free option called Rectangle. Both of these let you snap your windows to different positions instantly with shortcuts.

Alfred (macOS)

MacOS's Spotlight feature ( ⌘ + space ) is great for quickly opening apps, doing calculations, and many other tasks. Alfred is Spotlight on steroids. It's more powerful and highly customizable. Alfred can hook into your terminal and password manager, search specific sites, and much more. I can't recommend it enough. The paid version also gives you access to your clipboards history, so you can still paste text after copying other things.

iTerm2, Powerlevel10k, and (maybe) Fish (macOS)

While many beginners are intimidated by the command line (I certainly was) you'll learn to love its power and efficiency. The command-line -- often also called a shell or terminal -- lets you control your computer with text commands. MacOS uses Zsh or Bash which provide very similar commands and languages for scripting commands. If you are feeling adventurous, you can try Fish an alternative to Bash and Zsh that provides a few more modern features. You can easily change your default shell with one command, so it's low risk to try it out.

Customizing your terminal set up is a great way to make it more accessible as you are learning. The first thing to do is to install iTerm2 which is similar to Terminal but more powerful. Next, you'll want to customize the theme. This can be confusing to set up. Oh My Zsh is a popular way to do this, but I strongly recommend Powerlevel10k. It's the best looking, the most customizable, and has the best tutorial to make the setup process easier.

Pock (if your Mac has a Touch Bar)

If you've got a Touch Bar, you probably find it to be useless. Pock is a free app that lets you customize your Touch Bar and make it useful.
